# About

This project aims to solve Rosalind's problems and package them in a command-line interface (CLI) format.

## Repo structure

rosalind-cli/  
├── Cargo.toml           
├── src/  
│   ├── main.rs           
│   ├── lib.rs          
│   └── problems/       
│       ├── mod.rs      
│       ├── prob1.rs     
│       ├── prob2.rs     
│       └── ...         
└── README.md

## Problems solved

- [DNA](https://rosalind.info/problems/dna/);
- [RNA](https://rosalind.info/problems/rna/);
